–£–º–Ω–æ–∂–∏–º –∏—Å—Ö–æ–¥–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –Ω–∞ $-2 y^{-3}$: $-2 y^{-3} y' = -2x^3 + 2x y^{-2}$
–ü—Ä–∏ $z = y^{-2}$ –∏ $z' = -2 y^{-3} y'$, –ø–æ–ª—É—á–∞–µ–º: $z' = -2x^3 + 2x z \Rightarrow z' - 2x z = -2x^3$

### –†–µ—à–∞–µ–º –ª–∏–Ω–µ–π–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ

–ò–Ω—Ç–µ–≥—Ä–∏—Ä—É—é—â–∏–π –º–Ω–æ–∂–∏—Ç–µ–ª—å $\mu(x) = e^{\int (-2x) dx} = e^{-x^2}$:  $(e^{-x^2} z)' = -2x^3 e^{-x^2} \Rightarrow e^{-x^2} z = \int -2x^3 e^{-x^2} dx$

–ó–∞–º–µ–Ω–∞ $t = x^2$ –∏ $dt = 2x dx$: $\int -2x^3 e^{-x^2} dx = \int -x^2 \cdot 2x e^{-x^2} dx = -\int t e^{-t} dt$
$\int t e^{-t} dt = -t e^{-t} - e^{-t} + C \Rightarrow -\int t e^{-t} dt = t e^{-t} + e^{-t} + C = (x^2 + 1) e^{-x^2} + C \Rightarrow e^{-x^2} z = (x^2 + 1) e^{-x^2} + C \Rightarrow z = x^2 + 1 + C e^{x^2}$

### –ü–æ–¥—Å—Ç–∞–≤–ª—è–µ–º –∑–∞–º–µ–Ω—ã –æ–±—Ä–∞—Ç–Ω–æ

$z = y^{-2}$: $y^{-2} = x^2 + 1 + C e^{x^2}$  

–ü—Ä–∏ $y(0) = 1$ –ø–æ–ª—É—á–∞–µ—Ç—Å—è: $1 = 0 + 1 + C \cdot 1 \Rightarrow C = 0$ (–∫–∞–∫ —É–¥–æ–±–Ω–æ)

**–ò—Ç–æ–≥–æ**: $y(x) = \dfrac{1}{\sqrt{x^2 + 1}}$, $x \in \mathbb{R}$.

–ü—Ä–æ–≤–µ—Ä—è–µ–º: $y'(x) = \dfrac{-x}{(x^2 + 1)^{3/2}}$. $x^3 y^3 - xy = \dfrac{x^3}{(x^2 + 1)^{3/2}} - \dfrac{x}{\sqrt{x^2 + 1}} = \dfrac{x^3}{(x^2 + 1)^{3/2}} - \dfrac{x (x^2 + 1)}{(x^2 + 1)^{3/2}} = \dfrac{-x}{(x^2 + 1)^{3/2}}$.
